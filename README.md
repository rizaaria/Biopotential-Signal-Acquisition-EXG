# Biopotential Signal Acquisition EXG

## Overview

Biopotential Signal Acquisition EXG is an open-source biomedical instrumentation project developed to acquire and visualize electrophysiological signals such as Electrocardiography (ECG), Electromyography (EMG), Electroencephalography (EEG), and Electrooculography (EOG).

This project was developed by studying and reverse engineering the open-source BioAmp EXG Pill platform from Upside Down Labs. The work involved schematic analysis, PCB redesign, PCB fabrication, manual SMD assembly, hardware validation, and signal acquisition testing.

The objective of this project was to gain practical experience in biomedical signal acquisition hardware design and analog front-end development for biosignal monitoring applications.

## Reference Platform

This project was inspired by and developed based on the open-source BioAmp EXG Pill platform by Upside Down Labs.

Original Project:
https://github.com/upsidedownlabs/BioAmp-EXG-Pill

> Note: The hardware in this repository was independently redesigned and assembled. Signal visualization and filtering software used during testing are based on the original Upside Down Labs ecosystem.

## Key Contributions

- Reverse engineered the BioAmp EXG Pill hardware architecture
- Redesigned the schematic from scratch
- Designed custom PCB layout
- Generated fabrication-ready Gerber files
- Performed manual SMD soldering and assembly
- Conducted hardware debugging and validation
- Acquired ECG, EMG, EEG, and EOG signals
- Evaluated signal quality through experimental testing

## Features

|------------|----------|
| Analog Front-End IC | Biopotential signal amplification |

| Microcontroller | Data acquisition |
| USB Interface | Data communication |

## Recorded Signals

### Electrocardiography (ECG)

Recording electrical activity generated by the heart.

### Electromyography (EMG)

Recording electrical activity produced by skeletal muscles.

### Electroencephalography (EEG)

Recording electrical activity generated by the brain.

### Electrooculography (EOG)

Recording eye movement signals.

## Validation Results

The developed hardware successfully acquired:

- ECG waveforms
- EMG muscle activity signals
- EEG brain activity signals
- EOG eye movement signals

Signal visualization and filtering were performed using software provided by Upside Down Labs.
